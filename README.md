# WebPos React (Vite + Tailwind + Chart.js)

Proyecto React generado a partir de los HTML de login, dashboard, alertas, reportes, tablas y personal.

## Requisitos
- Node.js 18+

## Primeros pasos
```bash
npm install
npm run dev
```

## Rutas
- `/login` (inicio de sesión)
- `/dashboard`
- `/alertas`
- `/reportes`
- `/tablas`
- `/personal`

## Notas
- La autenticación es de demostración (sin backend).
- Gráficas con Chart.js replican el contenido de los mockups.
- Tailwind ya está configurado.
